# example-hello-world

run consloe.log('Hello World!!')

## install

```
$ npm i @rm-labo/example-hello-world
```

## usage

```
// ECMAScript
import hello from '@rm-labo/example-hello-world'
hello() // => Hello World!!


// CommonJS
const hello = require('@rm-labo/example-hello-world')
hello() // => Hello World!!

```
